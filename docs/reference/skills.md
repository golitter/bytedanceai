### taste-skill

https://github.com/Leonxlnx/taste-skill

| skills名称                 | 描述                                                         |
| -------------------------- | ------------------------------------------------------------ |
| design-taste-frontend      | **全能高级前端**：默认的通用型技能，产出高质量的前端代码，且不局限于某一种单一的视觉风格，适合绝大多数要求“质感”的页面。 |
| redesign-existing-projects | **项目重构优化**：针对已有项目。先审查现有的 UI，然后修复布局、间距、视觉层级和样式问题，适合旧项目翻新。 |
| high-end-visual-design     | **高端柔和视觉**：精致、平静、具有“昂贵感”的 UI。特点是更柔和的对比度、大量留白、高级字体以及弹性动效，适合奢侈品牌或高端SaaS。 |
| full-output-enforcement    | **防偷懒强制输出**：专治 AI 写代码写一半的问题。强制模型输出完整的代码，绝不允许出现 `// 此处省略...` 或 `// 其余代码同上` 等占位符注释。 |
| minimalist-ui              | **极简编辑风**：类似 Notion 或 Linear 风格的产品 UI。克制用色，结构清晰干脆，适合效率工具类产品。 |
